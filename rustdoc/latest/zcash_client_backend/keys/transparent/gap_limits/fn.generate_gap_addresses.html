Expand description

Generates transparent addresses to fill the gap for a given account and key scope.

This function queries the wallet backend (via AddressStore) to find the start of the first gap of unused addresses, then generates enough addresses to maintain the configured gap limit. If no gap exists (i.e., the address space is exhausted), this is a no-op.
